Data Analysis Code (Python)


import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

# Load quarterly earnings data
data = pd.read_csv('earnings.csv')

# Calculate growth rate and metrics
data['growth'] = data['earnings'].pct_change()
data['roi'] = (data['earnings'] / data['investment']) * 100

# Create professional visualization
sns.set_style('whitegrid')
plt.figure(figsize=(12, 6))
plt.plot(data['quarter'], data['earnings'], marker='o', linewidth=2)
plt.title('Quarterly Earnings Trend', fontsize=16, fontweight='bold')
plt.xlabel('Quarter')
plt.ylabel('Earnings ($M)')
plt.grid(True, alpha=0.3)
plt.show()

Financial Calculation (JavaScript)


// Calculate Return on Investment (ROI)
function calculateROI(profit, investment) {
    const roi = (profit / investment) * 100;
    return roi.toFixed(2);
}

// Calculate Compound Annual Growth Rate (CAGR)
function calculateCAGR(startValue, endValue, years) {
    const cagr = (Math.pow(endValue / startValue, 1 / years) - 1) * 100;
    return cagr.toFixed(2);
}

// Example usage
const q1Profit = 2500000;
const investment = 10000000;

Financial Formulas

Return on Investment (ROI):

\[ ROI = \frac{Net\ Profit}{Investment} \times 100 \]

Compound Interest:

\[ A = P(1 + \frac{r}{n})^{nt} \]

Where A is the amount, P is principal, r is rate, n is compounding frequency, t is time.
