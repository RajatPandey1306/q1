Python Code: Financial Analysis


def calculate_roi(profit, investment):
    """Calculate Return on Investment percentage"""
    return (profit / investment) * 100

def calculate_cagr(start_value, end_value, years):
    """Calculate Compound Annual Growth Rate"""
    return ((end_value / start_value) ** (1 / years) - 1) * 100

roi = calculate_roi(500000, 350000)
cagr = calculate_cagr(2500000, 3500000, 1)
print(f"ROI: {roi:.2f}% | CAGR: {cagr:.2f}%")

    quarter,
    SUM(revenue) as total_revenue,
    AVG(profit_margin) as avg_margin
FROM earnings_data
GROUP BY quarter
HAVING total_revenue > 2000000
ORDER BY quarter DESC;

Financial Formulas (MathJax)

Weighted Average Cost of Capital (WACC):

\[ \text{WACC} = \frac{E}{V} r_e + \frac{D}{V} r_d (1 - T) \]

Discounted Cash Flow (perpetuity with growth):

\[ \text{Value} = \frac{\text{FCF}_1}{r - g} \]

Earnings Per Share (EPS): \( \displaystyle EPS = \frac{\text{Net Income}}{\text{Weighted Avg. Shares}} \)

Statistical Formula

\[P(A|B) = \frac{P(B|A)P(A)}{P(B)}\]
